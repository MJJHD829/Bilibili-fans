# Bilibili Fans Counter

一个简单的 B 站粉丝数显示器。

## 部署说明

1. 部署到 Railway 平台
2. 访问 `你的域名/#用户ID` 来查看特定用户的粉丝数

```bash
project/
  ├── public/
  │   ├── clock-number.ttf
  │   ├── bilibili.png
  │   ├── styles.css
  │   └── index.html
  ├── main.js
  ├── package.json
  ├── .gitignore
  ├── manifest.json
  └── README.md 
```
